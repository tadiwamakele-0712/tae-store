# Tae Gift Hampers

Luxury gift hamper portfolio website for **Tae Gift Hampers** — Harare, Zimbabwe.

> *Say it better with us*

## Live site

- **GitHub Pages:** [tadiwamakele-0712.github.io/tae-store](https://tadiwamakele-0712.github.io/tae-store/)
- **Local:** open `index.html` in your browser

## Features

- Responsive product gallery (28 gift images)
- Category filters — Wedding, Corporate, Birthday, Celebration
- Lightbox viewer for full-size photos
- Dark / light theme toggle
- Contact form, WhatsApp, phone & email
- TikTok & Facebook social links
- Client testimonials section
- Mobile-friendly navigation

## Tech stack

- HTML5
- CSS3 (custom properties, grid, responsive)
- Vanilla JavaScript (no frameworks)

## Project structure

```
tae store/
├── index.html      # Main page
├── styles.css      # Styling + dark mode
├── script.js       # Gallery, lightbox, filters, theme
├── products.js     # Product data & categories
└── images/         # Logo & gift photos
```

```powershell

git add .

git push origin main
```

In repo **Settings → Pages**, set source to **main** branch, folder **/ (root)**.
